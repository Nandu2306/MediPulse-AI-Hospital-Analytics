# MediPulse AI – Hospital Operations Analytics

## Project Overview

An end-to-end hospital analytics project built using MySQL, SQL, Python, and Power BI to analyze hospital operations, billing, appointments, and treatments.

## Technologies Used

- MySQL
- SQL
- Python
- Pandas
- NumPy
- Plotly
- Power BI

## Features

- Hospital relational database
- SQL analytics
- Data cleaning
- Exploratory Data Analysis
- KPI Dashboard
- Interactive Power BI reports
